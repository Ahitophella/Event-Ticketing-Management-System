# Event Ticketing Management System

A database-driven application built on the **Oracle APEX** platform designed to handle event registration, ticketing workflows, and secure attendee tracking.

## 🚀 Key System Features
* **Database Logic:** Utilizes custom PL/SQL procedures, functions, and relational database triggers to maintain transactional integrity.
* **Component Architecture:** Configured interactive data reports and secure form submissions directly within the APEX environment.
* **Relational Design:** Structured mapping schemas designed to avoid data anomalies.

## 🛠️ Tech Stack
* **Platform:** Oracle APEX
* **Language:** SQL, PL/SQL
